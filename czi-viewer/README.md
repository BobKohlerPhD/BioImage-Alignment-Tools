# CZI Viewer

A portable napari-based CZI viewer.

This tool provides a stable, portable environment for viewing CZI files with napari and relevant plugins.

## Features
- Portable setup using micromamba (no system install needed)
- Optimized for CZI compatibility
- Includes key napari plugins (aicsimageio, animation, assistant, threedee)
- Pinned versions for stability (tifffile, imagecodecs)

## Installation & Usage
Run `Portable_CZI_Viewer.command` to start the setup and launcher.

On macOS, you may need to grant permission to run the script.
